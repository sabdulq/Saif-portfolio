About me

Analysis you can act on.

I'm a Data Science graduate from Indiana University–Purdue University Indianapolis (IUPUI), specializing in data analysis and business intelligence. What started as a childhood fascination with code grew into a career built on one belief: the right analysis, communicated clearly, helps people make better decisions.

I work across the full analytics lifecycle — validating and modeling data, building responsive dashboards, and translating findings for both technical and non-technical audiences. I care as much about clean documentation and trustworthy data as I do about the insight itself.

Lately I've been folding generative-AI tools into my workflow to accelerate exploratory analysis, documentation, and reporting — so the time goes where it matters: the thinking.
